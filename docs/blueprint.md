# **App Name**: Data Skill Builder

## Core Features:

- SQL Database Practice: Users can create and manage SQL schemas and practice querying data for simulated hotel and clinic management systems.
- Spreadsheet Data Analysis Tools: Provides an interface for performing data lookups and time-based analysis, mimicking spreadsheet functionality.
- Python Programming Environment: An integrated environment for writing and testing Python scripts focused on data conversion and string manipulation.

## Style Guidelines:

- Primary color: A deep navy blue (#1A237E) for actions and branding, conveying professionalism.
- Background color: A clean, light gray (#F5F5F5) to provide a neutral canvas for data and code.
- Accent color: A vibrant green (#4CAF50) used for success messages, highlights, and active states.
- Body text font: 'Open Sans' for its clear readability across various screen sizes.
- Code and data display font: 'Roboto Mono' for excellent legibility of code snippets and tabular data.
- Utilize simple, functional Material Design icons for common actions like save, run, and export.
- Structured, multi-panel layout, allowing users to view code, output, and data simultaneously for efficient workflow.
- Responsive design ensuring optimal display and interaction on both desktop and tablet devices.
- Subtle animations for loading states, script execution, and user feedback to enhance the interactive experience without distraction.
